WOG Telugu Bible App

World Of God Telugu Bible mobile application.

Features

- Offline Telugu Bible
- Book → Chapter → Verse navigation
- Verse highlight
- Verse search
- Bookmark verses
- Share verses
- Dark mode
- Font size adjustment

Bible Source

XML format Telugu Bible stored in:

assets/bible/Bible.xml

Structure:

XMLBIBLE
└ BIBLEBOOK
└ CHAPTER
└ VERS

Project Structure

lib/

screens/
models/
services/
widgets/

assets/bible/Bible.xml

Build APK

GitHub Actions automatically builds APK when code is updated.

Steps:

1. Edit code
2. Commit changes
3. GitHub Actions builds APK
4. Download APK from Actions → Artifacts
